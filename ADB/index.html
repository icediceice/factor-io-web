Welcome

This guided setup unlocks optional Blink features that require a one-time connection from your computer. All steps are optional — skip anything you don't need.

What you'll set up:

  1. Banking Mode & Auto-Recovery — keeps screen protection alive
  2. Screen Protection Service — auto-intercept when protected apps open
  3. Banking Mode — auto-pause accessibility for banking apps

Requirements: Chrome or Edge on desktop, USB cable, phone unlocked.

Enable USB Debugging

Enable USB Debugging on your Android phone so your computer can communicate with it.

  1. Open Settings → About phone
  2. Tap Software information (Samsung) or similar
  3. Tap Build number 7 times to unlock Developer Options
  4. Go back to Settings → Developer options
  5. Enable USB debugging
  6. Connect your phone with a USB cable
  7. In the notification bar tap the USB connection and select File Transfer (MTP)
  8. On your phone tap Allow on the USB debugging prompt
Using Android Studio or ADB on this computer? Stop the ADB server first, otherwise it blocks WebUSB.
Open a terminal and run: adb kill-server

Connect Your Phone

Click Connect and select your phone from the browser's USB device picker.

Waiting for connection…

Or: Run Commands Manually

If WebUSB isn't working, paste these commands into a terminal with ADB on your PC.

Still not connecting? Unplug and replug the USB cable, then click Retry Connection. The phone's ADB interface sometimes needs a reset after the server is stopped.
adb shell pm grant com.blink.app android.permission.WRITE_SECURE_SETTINGS adb shell settings put secure enabled_accessibility_services \ com.blink.app/com.blink.intercept.BlinkAccessibilityService

Grant Permissions

Each step is optional. Grant what you need, skip the rest.

Banking Mode & Auto-Recovery Free

Grants Blink the ability to manage Android's accessibility service list. Required for two features: Banking Mode (auto-pauses the screen protection service while banking apps are open) and A11y self-heal (silently re-enables screen protection if an OEM or system update revokes it).

Screen Protection Service Free

Instantly detects when protected apps are opened and covers them before content is visible. Uses Android's Accessibility Service.

Banking Mode Free Requires Step 1

Automatically pauses accessibility services when you open banking apps so they work without interruption. Services are restored when you close the app.

Setup Complete

Your granted permissions are now active. A few final steps:

  1. Disconnect the USB cable
  2. In Developer Options, disable USB Debugging (recommended for security)
  3. Open Blink and verify each feature works as expected
  4. For Banking Mode: go to Protected Apps in Blink and tap the bank icon next to each banking app you want protected
Setup complete.
